# Code Availability

## Overview
This repository contains the code used to process and prepare the dataset:
Global Personal Networks Datset [Mapping Cultural Diversity through Personal Networks] [10.5281/zenodo.19134491]

## Files
- `clean4publication.py`  
  Cleaning, harmonization, ID construction, and filtering  

- `figures4publication.py`  
  Generation of summary tables and figures  

## Packages
- Python: 3.13.1
- pandas: 2.2.3
- numpy: 2.1.0
- matplotlib: 3.10.0
- seaborn: 0.13.2

## Notes
- Designed for Qualtrics survey exports with ego-network structure  
- Paths are environment-specific and must be adapted
